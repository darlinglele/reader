darlinglele/reader
===============
基于darlinglele/mobile-app-seed, 个人手机端阅读器

Web环境安装:

1. git clone https://github.com/darlinglele/reader.git

2. 安装 Nodejs

启动http-server :

1. cd reader

2. npm start

浏览:

1. http://localhost:8000/developer.html





Android手机安装
 
0. npm start

1. cordova platform add android

2. cordova run -device
